HUFFMAN ALGORITHM (FROM SCRATCH)

This commission-based C++ project tackles Huffman coding for compression and decompression. In compression, it efficiently encodes text files, storing binary data and a frequency table. The decompression module reads Huffman-encoded files, reconstructs the Huffman tree, and decodes binary data to restore the original text. What sets this project apart is its creation from scratch, using only the iostream and fstream libraries. It integrates priority queues and custom string functions for efficiency, providing a comprehensive Huffman coding solution.
